From paper to NetLogo.

This site helps first-time users learn NetLogo Web using the toy example from Zollman's "The communication structure of epistemic communities". The goal is to formalize the toy example, learn how interface widgets connect to code, and then run the two-agent model.

Course flow

Read the paper first, then move through the pages in order:

  • Page 1 formalizes states, actions, results, and beliefs.
  • Page 2 shows the minimal NetLogo code and manual widget setup.
  • Page 3 implements the two-agent experiment in NetLogo.
